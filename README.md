# Forever Pets

[![](https://badges.moddingx.org/modrinth/versions/forever-pets)](https://modrinth.com/mod/forever-pets)

Simply makes all tamed cats, wolves and parrots completely immortal. Even /kill won't kill them.

If you are playing multiplayer, you only need the mod on your server.

If you already have tamed pets, they will become immortal once you install the mod.

Inspired by [Immortal Pets](https://modrinth.com/mod/immortal_pets) which does the exact same thing, except is unfortunately outdated and doesn't have source available.
